Stories of The Land
Longterm IT Development Project:

Developed an interactive ASP.NET web site for a project called Stories of The Land. Stories of The Land is a walking curriculum that uses the Trail through Hannin Creek camp. Using QR codes located near plants native to the boreal forest, the user can connect to a website that contains identifying information about the plant. The website also included Cree names and pronunciation audio and traditional Indigenous uses or cultural significance for plants.
